## Local Development

The following assumes the use of `node@>=10`.

### Install Dependencies

`yarn`

### Compile Contracts

`yarn compile`

### Run Tests

`yarn test`

## Deployments

Run `yarn deploy-polygonAmoy`or `yarn deploy-sepolia`
